DOM refers to DOCUMENT OBJECT MODEL which is a kind of programming interface for web documents. It comprises the structure and content of a document on the web. It also can change the document structure, style and content.It shows the document as nodes and objects.

In 5 ways we can select element from DOM (a) getElementsByTagName() (b) getElementsByClassName() (c) getElementById() (d) querySelector() (e) querySelectorAll()

A concept in the DOM(Document Object Model) is the event bubble. When an element receives an event and that event bubbles up to its parent and ancestor elements in the dom tree until it gets to the root element.
